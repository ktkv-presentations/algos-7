Встроенные методы и функции

Индексация строк


s = "Python"
# P y t h o n
# 0 1 2 3 4 5
print(s[0])  # P
print(s[-1]) # n
При положительном индексе мы получаем n символ с начала строки, а при отрицательном с конца строки

Срезы строк


s = "Python"
# P y t h o n
# 0 1 2 3 4 5
print(s[0:3])  # Pyt
print(s[2:6])  # thon

print(s[:3])   # Pyt
print(s[2:])   # thon

Длина строки


s = "Python"
len(s) # 6

Методы изменения строк

lower()

преобразовывает строку в нижний регистр

upper()

преобразовывает строку в верхний регистр


s = "Python"
print(s.lower())  # python
print(s.upper())  # PYTHON

Методы изменения строк

capitalize()

делает первую букву строки заглавной

title()

делает заглавными первые буквы каждого слова


s = "hello world"
print(s.capitalize())  # Hello world
print(s.title())       # Hello World

Методы изменения строк

swapcase()

меняет регистр символов строки на противоположный


s = "PyThOn"
print(s.swapcase())  # pYtHoN

Работа с числовым кодом

ord()

возвращает числовой код символа (его позицию в таблице Unicode).

chr()

возвращает символ по его числовому коду


print(ord('A'))  # 65
print(chr(65))   # 'A'

Методы для работы с содержимым строк

startswith()

проверяет начинается ли строка с указанной подстроки

endswith()

проверяет заканчивается ли строка с указанной подстроки


s = "Python programming"
print(s.startswith("Py"))  # True
print(s.endswith("ing"))   # True

Поиск по строке

find()

возвращает индекс первого вхождения подстроки или -1, если подстрока не найдена

rfind()

возвращает индекс последнего вхождения подстроки или -1, если подстрока не найдена

index()

работает так же как и find, но вызывает ошибку ValueError, если подстрока не найдена

rindex()

работает так же как и rfind, но вызывает ошибку ValueError, если подстрока не найдена 


s = "Python programming"
print(s.find("pro"))    # 7
print(s.find("xyz"))    # -1
print(s.index("pro"))   # 7

Счет по строке

count()

возвращает количество вхождений подстроки в строке


s = "banana"
print(s.count("a"))  # 3

Замена в строке

replace()

заменяет все вхождения одной подстроки на другую

Методы проверки содержимого строки

isalpha()

возвращает True, если все символы в строке являются буквами

isdigit()

возвращает True, если строка состоит только из цифр

isalnum()

возвращает True, если строка содержит только буквы и цифры

isspace()

возвращает True, если строка состоит только из пробельных символов (например, пробел, табуляция)

Методы проверки содержимого строки

islower()

возвращает True, если строка состоит только из символов нижнего регистра

isupper()

возвращает True, если строка состоит только из символов верхнего регистра

Методы работы с пробелами

strip()

удаляет пробелы с обоих концов строки

lstrip()

удаляет пробелы только слева

rstrip()

удаляет пробелы только справа

Методы разделение и соединения строк

split()

разбивает строку на подстроки по указанному разделителю и возвращает список

join()

объединяет список строк в одну строку с указанным разделителем


s = "Python is fun"
words = s.split()
print(words)           # ['Python', 'is', 'fun']
print(words.join("-")) # "Python-is-fun"

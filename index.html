Строковые методы и функции

Индексация строк


s = "Python"
# P y t h o n
# 0 1 2 3 4 5
print(s[0])  # P
print(s[-1]) # n
При положительном индексе мы получаем n символ с начала строки, а при отрицательном с конца строки

Срезы строк


s = "Python"
# P y t h o n
# 0 1 2 3 4 5
print(s[0:3])  # Pyt
print(s[2:6])  # thon

print(s[:3])   # Pyt
print(s[2:])   # thon
